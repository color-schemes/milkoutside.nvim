# MilkOutside Steam Theme

A dark Steam theme based on the [milkoutside.nvim](https://github.com/tiagovla/milkoutside.nvim) colorscheme with cosmic-inspired colors.

## Colors

The theme uses the milkoutside color palette:

- **Background**: Dark cosmic colors (`#040607`, `#1a1e2e`)
- **Foreground**: Light gray text (`#e8e8e8`)
- **Primary Blue**: `#63c3dd`
- **Accent Colors**: Various cosmic-inspired colors including magenta, orange, yellow, and green

## Installation

1. Download the [latest release](https://github.com/tiagovla/milkoutside.nvim/releases/latest)
2. Extract the `steam` folder from the `extras` directory
3. Navigate to your Steam installation folder
4. Place the extracted `steam` folder in `steamui/skins/`
5. Enable JavaScript support in your Millennium patcher
6. Select the "milkoutside" theme in your Millennium settings

## Requirements

- [Millennium](https://github.com/SteamClientHomebrew/Millennium) Steam patcher
- JavaScript support enabled in Millennium

## Features

- Complete Steam client theming
- Consistent dark aesthetic across all Steam windows
- Cosmic-inspired color palette
- Optimized for both light text on dark backgrounds
- Custom scrollbars and UI elements
