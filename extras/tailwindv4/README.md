# MilkOutside for Tailwind CSS v4

The MilkOutside theme as Tailwind CSS theme color variables. Compatible with version 4 of Tailwind.

## Usage

1. Copy the theme CSS file(s) to your project.

2. Import in the CSS file to be processed by Tailwind CSS.

   ```css
   /* styles.css or whatever your CSS file is named */
   @import "tailwindcss";

   @import "./milkoutside.css";
   @import "./milkoutside.css";
   @import "./milkoutside.css";
   @import "./milkoutside.css";
   ```

3. Use the theme's colors where ever you would normally reference colors.

   ```html
   <button class="border border-milkoutside-border bg-milkoutside-bg text-milkoutside-fg">
     Click me!
   </button>
   ```
